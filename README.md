# Airline Management System

A Java-based Airline Management System that simulates basic airport operations using core Data Structures.

## Project Description

This project is a desktop application built using Java (Swing) that manages airline operations such as:

- Managing flights
- Booking tickets
- Cancelling bookings
- Viewing available flights
- Viewing passengers per flight
- Adding new flights dynamically

The system uses custom implementations of:
- Linked List (for flights management)
- Queue (for passengers per flight)

## Features

### 1. Flight Management
- Add new flights
- Display all available flights
- Update flight departure time
- Search and delete flights

### 2. Booking System
- Book a flight for a passenger
- Choose seat type (VIP / Economy)
- Choose payment method
- Auto seat management per flight
- Ticket generation

### 3. Cancellation System
- Cancel booking using Passenger ID
- Restore available seats

### 4. Passenger Management
- Each flight maintains its own passenger queue
- View passengers per flight

### 5. GUI Interface
- Built using Java Swing
- CardLayout navigation
- Login screen
- Menu-based system

## Technologies Used

- Java
- Java Swing (GUI)
- Object-Oriented Programming (OOP)
- Data Structures (Linked List, Queue)

## Project Structure

- AirlineGUI.java → Main GUI interface
- Flightsystem.java → Console version (optional)

## How to Run

1. Open project in NetBeans or IntelliJ IDEA
2. Make sure Java JDK is installed
3. Run:
   - AirlineGUI.java (for GUI version)
   - OR Flightsystem.java (for console version)
